# Contract Finder – Gov Opportunities Dashboard

Next.js dashboard for government contract opportunities with a 20% sidebar and 80% main content area.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Highcharts** (charts)
- **AG Grid** (tables)

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- **Dashboard** (`/`) – Cards, column chart (opportunities by agency), area chart (weekly trend), and opportunities table.
- **Sam.Gov** (`/sam-gov`) – Placeholder.
- **Septa** (`/septa`) – Placeholder.
- **Recommendations** (`/recommendations`) – Placeholder.

## Layout

- Sidebar: 20% width, dark slate.
- Main: 80% width, light slate background.
- Accent: blue (#0c8ee6) for links and chart series.
